# cub3D
Cub3D is a 3D game with C language implemented using ray-casting

How to use :

make 

./cub3D map.cub

map :

NO north texture (xpm file)
SO south texture (xpm file)
WE west texture (xpm file)
EA north texture (xpm file)                  
S  sprite texture
F  floor color RGB
C  Ceiling color RGB
R  W H
